# TableFinder Pro - Site Improvements Plan

## Goal
Improve the TableFinder Pro site with better UX, accessibility, missing features, and polish.

## Phases
1. **Core UX Fixes** — Fix broken list view toggle, improve mobile experience
2. **Accessibility & SEO** — ARIA labels, keyboard nav, meta tags, Open Graph
3. **New Features** — Reservation status, table booking flow, enhanced search
4. **Visual Polish** — Animations, loading states, micro-interactions
5. **Admin Enhancements** — Server assignment, table notes, history log
